How To Play

  • Each number represents consecutive filled (black) cells.
  • Multiple numbers mean separate groups.
  • There must be at least one blank cell between groups.
  • Left numbers → Row clues. Top numbers → Column clues.

Controls

  • Left Click → Fill cell
  • Right Click → X mark
  • Undo → Reverse last move
  • Clear Grid → Reset puzzle

Competition Rules

  • Timer starts when game begins.
  • Board locks after correct submission.
  • 52-character secure code generated on success.
